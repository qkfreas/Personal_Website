This one, the one you, the user, are viewing is a website inspired by a Summer 2018 work project.

Description
Using JavaFX as the UI, this project utilizes several different parts of OOP.
The concept of the this project is a simple card game. The user clicks Deal and two cards are placed on the game board, one for the user and one for the computer. The cards are compared and the highest card wins.
